SafeOnline SDK

This document contains the API documentation for the SafeOnline SDK. The SafeOnline SDK contains Java components to ease the communication with the SafeOnline services. These services are:

  • Authentication web application: can be used by an application to authenticate a user.
  • Attribute web service: can be used by applications to retrieve attributes from subjects that confirmed the corresponding application identity.
  • Data web service: can be used by applications to perform CRUD operations on subject attributes if they are assigned the attribute provider role.
  • Name Identifier Mapping web service: can be used by application to map a username to the corresponding user identifier.
Besides client components for these web service you'll also find various components in this SDK to ease integration of the SafeOnline authentication mechanism within servlet container based web applications.
