README for Siemens WS Authentication Acceptance Test Console
============================================================

=== 1. Overview

This ZIP contains the Siemens WS Authenticaiton Acceptance Test Console.


=== 2. Installation

Just unpack the ZIP.


=== 3. Running

On Unix/Linux machines you can start the application via the following command:
./acceptance-console.sh
