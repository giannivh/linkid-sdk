README for OLAS Application Console
===================================

=== 1. Overview

This ZIP contains the OLAS Application Console.


=== 2. Installation

Just unpack the ZIP.


=== 3. Running

On Unix/Linux machines you can start the application via the following command:
./app-console.sh
